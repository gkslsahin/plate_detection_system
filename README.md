# plate_detection_system
Openalpr based plate recognition system
required library<br/>
-openalpr<br/>
-opencv<br/>
-tesseract<br/>
plate recognition on picture<br/>
cd alpr_img<br/>
./build.sh<br/>
./plate_image image<br/>

plate recognition on video stream<br/>
cd stream<br/>
./build.sh<br/>
./plate_video_stream<br/>
